
# Task-Management-App

This is a simple Task Management App built with Next.js, TypeScript, Chakra UI, and GraphQL Yoga. It supports CRUD (Create, Read, Update, Delete) operations, allowing users to manage their tasks efficiently.

## Features

- **Add Tasks**: Create new tasks to stay organized.
- **Edit Tasks**: Update task details with ease.
- **Delete Tasks**: Remove tasks when they’re no longer needed.
- **Mark Tasks as Completed**: Track your progress by marking tasks as complete.
- **Separate Pages for Completed Tasks**: View completed tasks on a dedicated page.
- **Live Updates**: Real-time updates for task changes.
- **Task Details Modal**: Click on a task's edit button to view or edit task details in a modal.

## Tech Stack

- **Frontend**: Next.js, TypeScript, Chakra UI
- **Backend**: GraphQL Yoga, MongoDB
- **State Management**: Context API
- **Animation**: Framer Motion
- **Subscriptions**: subscriptions-transport-ws for real-time updates
